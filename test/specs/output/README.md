# API 규칙 저장소

이 폴더는 AI 에이전트를 위한 API 호출 규칙들을 포함하고 있습니다.

## 📄 주요 파일 안내
- **[agent.md](./agent.md)**: AI 에이전트가 API를 구현할 때 따라야 할 상세 지침서
- **[llms.txt](./llms.txt)**: LLM이 전체 API 구조를 빠르게 파악하기 위한 토큰 효율적 인덱스

## 🔗 도메인별 규칙
모든 규칙은 도메인별 폴더에 분류되어 저장되어 있습니다.
전체 API 목록과 파일 경로는 **[llms.txt](./llms.txt)**를 참조하세요.
